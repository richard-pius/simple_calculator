# Kotlin Compose Calculator üì±

A sleek, modern calculator app built with **Android Jetpack Compose**, following the **MVI (Model-View-Intent)** architectural pattern. This project was developed as a learning exercise based on the tutorial by Philipp Lackner.

---

## ‚ú® Features

* **Full Arithmetic Support:** Addition, Subtraction, Multiplication, and Division.
* **MVI Architecture:** Clear separation of concerns between UI state and business logic.
* **Custom UI:** Built entirely with Composable functions, featuring a dark-themed design.
* **Edge-to-Edge:** Utilizes modern Android window insets for a full-screen, immersive experience.
* **Responsive Layout:** Uses weights and aspect ratios to ensure the keypad scales perfectly on any screen size.

---

## üõ†Ô∏è Tech Stack

* **Language:** [Kotlin](https://kotlinlang.org/)
